# Roadmap
Este roadmap representa la planificación actual del proyecto y podrá modificarse durante el desarrollo en función de nuevas ideas, cambios de diseño o prioridades.

No contiene tareas específicas ni bugs. Para eso se utilizará el tablero de Trello.

Cada Epic representa una etapa importante del desarrollo del juego y finaliza con una versión estable y jugable.

---

# Epic 1 — Blackjack Base

## Objetivo

Construir una base sólida del juego implementando las reglas clásicas del Blackjack y una arquitectura limpia sobre la cual desarrollar nuevas mecánicas.

## Entregables

- Blackjack completamente funcional.
- Dealer con comportamiento clásico.
- UI funcional.
- Arquitectura inicial del proyecto.
- Proyecto publicado en GitHub.

**Versión:** v0.1.0

Estado: Completado
