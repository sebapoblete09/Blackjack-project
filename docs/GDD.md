# 1. Visión del proyecto

Fantasy Blackjack es un videojuego de cartas para un jugador inspirado en las reglas clásicas del Blackjack.

El objetivo no es recrear un casino tradicional, sino utilizar el Blackjack como base para construir un juego de estrategia con progresión, decisiones y mecánicas propias dentro de un mundo de fantasía medieval (Ambientacion provisoria).

El jugador irá desbloqueando nuevas formas de jugar mediante reliquias, cartas especiales y modificadores que alterarán las reglas de cada partida.

No se descarta cambiar el blackjack de base por un nuevo juego de cartas similar, creado desde 0 pero influenciado en este.

---

# 2. Objetivos del proyecto

## Objetivo principal

Desarrollar un juego de cartas con identidad propia, donde el Blackjack sea el núcleo del gameplay, pero no el límite del diseño.

## Objetivos secundarios

- Aprender desarrollo de videojuegos utilizando Godot.
- Diseñar una arquitectura escalable.
- Publicar el proyecto en GitHub.
- Publicar versiones jugables durante el desarrollo.
- Finalizar un proyecto de principio a fin.

# 4. Pilares del juego

Los pilares representan las ideas que nunca deberían perderse durante el desarrollo.

## 1. Fácil de aprender

Las reglas básicas deben entenderse en pocos minutos.

## 2. Difícil de dominar

La profundidad vendrá de las decisiones y los modificadores, no de reglas complicadas.

## 3. Cada partida debe sentirse diferente

Las reliquias, eventos y modificadores deberán generar nuevas estrategias en cada run.

## 4. La estrategia debe ser más importante que la suerte

Aunque exista azar, el jugador debe sentir que sus decisiones tienen impacto.

# 6. Alcance inicial

La primera versión del proyecto buscará completar un Blackjack clásico completamente funcional antes de comenzar a incorporar nuevas mecánicas.

No se añadirán sistemas complejos hasta tener una base sólida.
